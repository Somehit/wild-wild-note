# wild-wild-note
## The good, the bad and the note-taking app
This project aims to create a powerful and flexible note-taking application that remains simple for the average user. Wild Wild Notes delivers a premium note-taking experience that is local-first, cross-platform, end-to-end encrypted (E2EE), and open-source.

## Made for liberty 
This project is designed for anyone who wants a capable, powerful notes app that stays simple and frictionless to use. It is not built for tinkerers who want to endlessly tweak their workspace, but for people who want a clean, private space for their mind.

## One note, many views
Wild Wild Notes uses a very simple principle: one note, many views. Every note sits at the same level, but you can access them through different lenses—notebooks, tags, custom views, favorites, archives, and more. A powerful search engine makes finding any note effortless. This approach is designed to simplify capture and improve discovery.
